# URL Shortener MicroSaaS PRD

## Project Overview
- URL Shortener MicroSaaS - A platform to create shortened URLs
- Target users: General users who need to shorten URLs for sharing

## Core Features
1. URL Shortening - Create short URLs from long URLs
2. User Authentication - Login and register accounts
3. Dashboard - Manage shortened URLs
4. Premium Plans - Paid subscription for advanced features

## User Interactions
- Enter long URL and get shortened version
- Register/Login to manage URLs
- View statistics for shortened URLs
